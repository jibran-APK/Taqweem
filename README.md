# Taqweem
Taqweem is a beautifully designed calendar application that combines a custom Hijri calendar system with traditional lunar information, anakbintang guidance and daily fortune guidance in one easy-to-use app.
Whether you're tracking Hijri dates, following moon-day traditions, or simply looking for daily inspiration, Taqweem provides a unique and meaningful experience.

Features

🌙 Custom Hijri Calendar

View dates based on the custom Taqweem calendar.
Accurate Gregorian and custom Hijri date display.
Smooth monthly navigation.
Today's date highlighted automatically.

⭐ Anak Bintang (Star Guide)

View the current moon day.
Explore traditional moon-day information.
Beautiful lunar visualization.

🔮 Daily Fortune

Receive a new daily fortune every day.
Inspirational guidance for love, career, health, family, and personal growth.
Positive messages to help start your day with confidence.

📅 Dual Calendar Display

Gregorian Calendar
Custom Hijri Calendar
Easy date conversion and navigation.

🎨 Simple & Elegant Design

Clean and intuitive interface.
Fast and lightweight performance.
Optimized for Android devices.
